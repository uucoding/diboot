1. 启动项目

1.1. 命令行启动

打开命令行，并进入项目目录下，运行命令即可启动开发环境的项目。

1.1.1. npm

npm install
npm run serve

1.1.2. yarn

yarn
yarn serve

1.2. UI界面启动

  • 打开命令行，在任意目录下，运行一下命令，即可打开vue的UI管理界面。
  • UI界面中先导入当前项目，然后可以在UI界面上进行提供的相关功能的操作等。常用功能包括vue项目管理，启动项目，打包，eslint语法检查，插件管理，依赖管理等。
vue ui
